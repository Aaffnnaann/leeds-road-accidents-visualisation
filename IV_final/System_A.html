System A
Spatial Exploration of Road Traffic Accidents
Dataset: Leeds Road Traffic Accidents — Kaggle Source ↗
Visualisation Tasks
T1
Explore spatial features [Search → Explore; Target: Feature] — Browse the map to discover geographic accident clusters and hotspots across Leeds.
T2
Discover temporal distribution [Analyse → Discover; Target: Distribution] — The Weekday × Hour heatmap reveals which day-hour combinations have the highest accident density.
T3
Select a subset and summarise severity [Search → Browse; Query → Summarise] — Drag the map and/or drag across hours to filter; the severity chart updates for the selected subset.
Interactions: Drag on the map to select a spatial region → filters heatmap & severity. Drag left-right on the heatmap to select hours → filters map & severity. Both brushes compose (bidirectional).

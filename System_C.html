System C
Condition-Centric Exploration of Road Traffic Accidents
Dataset: Leeds Road Traffic Accidents — Kaggle Source ↗
Visualisation Tasks
T1
Explore spatial features [Search → Explore; Target: Feature] — Browse the map to discover hotspots; it responds to condition and hour selections.
T2
Discover temporal distribution [Analyse → Discover; Target: Distribution] — The stacked area chart reveals how each road surface condition contributes to accidents across the day.
T3
Select a subset and summarise severity [Search → Browse; Query → Summarise] — Click a road surface, drag an hour range, and/or drag the map to filter; severity chart updates.
Bidirectional Hub: The map is the shared bidirectional hub — dragging it reshapes both the condition bar (top-left) and the stacked area chart (top-right) simultaneously. Condition bar ↔ Map and Hour area ↔ Map are both fully bidirectional.
